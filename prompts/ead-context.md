# CONTEXTO DEL PROYECTO EAD

Este es un sistema de educación a distancia (EAD).

## OBJETIVO

Gestionar propuestas académicas, usuarios e interesados.

## FUNCIONALIDADES PRINCIPALES

* Gestión de carreras/cursos
* Recolección de interesados
* Visualización de documentación
* Dashboard administrativo
* Sistema de notificaciones futuras (email)

## BASE DE DATOS

* En transición hacia PostgreSQL (antes JSON)

## USUARIOS

* Múltiples usuarios con perfiles y permisos

## UI/UX

* Estilo minimalista
* Dark mode por defecto
* Posibilidad de modo claro

## REGLAS IMPORTANTES
